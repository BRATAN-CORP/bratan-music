### Что важно знать следующему агенту
- **Размер index-файлов держим до ~300 строк.** Если раздел разрастается
  — выноси в подпапку с собственным `index.md`.
- **Wiki-ссылки** — формат `[[путь/index|подпись]]` для внутренних
  переходов. Для внешних (GitHub, код, MDN) — обычные markdown-ссылки.
- **Лог обязателен.** После каждой задачи добавляй запись в
  `docs/daily-changes/YYYY-MM-DD.md` по формату из
  [[index|daily-changes/index.md]].
- **Tracker'ы.** Для длительных рефакторингов заводи
  `docs/refactor/<имя>.md` (по запросу пользователя — первый такой
  файл должен быть создан перед началом UI/UX-рефакторинга).
- **Не дублируем README.** Если что-то есть в `README.md` — линкуем,
  не копируем. База знаний — про **где найти** и **зачем**, а не
  про маркетинг.
